# This script`s will work from **RHEL** && **CENTOS** only

## **This script`s run in SUDO PRIVILEGE**

### Make it executable install:
```bash
chmod +x install.sh
```
### Make it executable uninstall:

```bash
chmod +x remove.sh
```

### install 

```shell
sudo ./install.sh
```

(or)

```shell
sudo bash install.sh
```


### uninstall 

```shell
sudo ./remove.sh
```

(or)

```shell
sudo bash remove.sh
```
